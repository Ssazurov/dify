Настройка двухфакторной аутентификации Multifactor

Для настройки второго фактора аутентификации требуется установить и настроить MULTIFACTOR Radius Adapter.

Настройка MULTIFACTOR Radius Adapter описана в инструкции:

Настройка двухфакторной аутентификации на Solar SafeInspect

Данный раздел поможет настроить двухфакторную аутентификацию в системе Solar SafeInspect.

## Настройка внешнего сервера RADIUS

Для настройки внешнего сервера RADIUS выполнить пошагово (Рис. 1):

1. Перейти в раздел **Настройки** > **Внешние сервисы** > **Внешняя аутентификация**;
2. Нажать кнопку **Добавить сервер RADIUS**;
3. Заполнить поля для новой аутентификации:
Адрес: адрес компонента MULTIFACTOR RADIUS Adapter;

Порт;

Начальный таймаут: первоначальное время ожидания в секундах (по умолчанию равно 2);

Максимальный таймаут: максимальное время ожидания в секундах (по умолчанию равно 8);

Максимальное число попыток: по умолчанию равно 4;

Пароль: Shared Secret из настроек компонента;

Подтверждение пароля: повторно указать значение Shared Secret.

4. Нажать кнопку **Сохранит ь**;
5. Применить отложенные изменения на вкладке **Применить изменения **кнопкой **Применить**.
![](Solar_SafeInspectt_2.4.10_Инструкция_по_настройке_2FA_Multifactor_media\image001.png)

Рис. . Добавление новой аутентификации

## Настройка правила

Примечание:

> Совместимость Solar SafeInspect и **Multifactor** работает по протоколам **SSH** и **RDP**

Для настройки правила подключения необходимо:

1. Перейти в раздел **Политики** > **Правила**;
2. Нажать кнопку **Добавить правило **(Рис. 2);
3. ![](Solar_SafeInspectt_2.4.10_Инструкция_по_настройке_2FA_Multifactor_media\image002.png)
Рис. . Добавление правила

4. Указать **имя правила**;
5. Выбрать **коллектор**, который будет его обрабатывать
> Если коллекторы не указаны, правило будет обрабатываться всеми коллекторами.

6. В поле **Подключение с** выбрать протокол подключения (**RDP** или **SSH**). При необходимости добавить критерии обработки правила.
7. Нажать **Продолжить**.
8. В блоке **Аутентификация** в параметре **Аутентификация** **Клиент-Коллектор** выбрать значение **Аутентиф ицировать средствами сервера RADIUS**.
9. ![](Solar_SafeInspectt_2.4.10_Инструкция_по_настройке_2FA_Multifactor_media\image003.png)
Рис. . Настройка аутентификации

10. В появившемся параметре **Сервер RADIUS** выбрать значение ранее добавленного сервера.
11. ![](Solar_SafeInspectt_2.4.10_Инструкция_по_настройке_2FA_Multifactor_media\image004.png)
Рис. . Добавление сервера

12. Параметр **Проверка подлинности Коллектор-Назначение** можно выбрать исходя из целей:
13. Передача исходной пары **Логин: Пароль** на целевой сервер (**Ретранслировать пароль и ввод с клавиатуры**);
14. Включение сопоставления учетных записей (**Сопоставленные учетные данные**).
![](Solar_SafeInspectt_2.4.10_Инструкция_по_настройке_2FA_Multifactor_media\image005.png)

Рис. . Проверка подлинности

15. Заполнить остальные параметры, исходя из целей, или оставить значения по умолчанию.
16. Нажать кнопку **Сохранить**.
17. Применить отложенные изменения на вкладке **Применить изменения** кнопкой **Применить**.
На этом настройка правила подключения завершена.

Теперь в подключениях, где требуется второй фактор, будет появляться диалоговое окно для ввода второго пароля.

![](Solar_SafeInspectt_2.4.10_Инструкция_по_настройке_2FA_Multifactor_media\image006.png)

Рис. . Диалоговое окно
