В случае загрузки обновления с сайта компании или установки апгрейда через веб-интерфейс Solar SafeInspect (файлы с расширением *.gpg), проверка дистрибутива и контрольных сумм файлов происходит автоматически. Если ПО выявит какие-либо несоответствия в ходе проверки, система обязательно выведет сообщение об ошибке. В этом случае установку ПО или апгрейда следует прекратить и обратиться в службу сервиса для получения консультации.

# Проверка контрольных сумм дистрибутива при использовании ОС Windows

## Проверка контрольных сумм дистрибутива с использованием утилиты CertUtil

Для определения целостности дистрибутива при использовании ОС Windows 7 и новее рекомендуется использовать утилиту **CertUtil**. Эта утилита входит в состав Windows 7 и новее.

Чтобы определить контрольные суммы дистрибутива необходимо:

1. Загрузить **образ диска** на локальное хранилище;
2. Запустить командную строку (открыть меню **Пуск** → найти приложение **Командная строка** или **Cmd** → нажать **Открыть**) (Рис. 1);
![](Solar_SafeInspect_Инструкция_по_проверке_КС_дистрибутива_media\image001.png)

Рис. 1. Командная строка

3. В командной строке ввести команду:
- для получения контрольной суммы **MD 5 **(Рис. 2):
certutil -hashfile "<путь_к_файлу>\<имя_файла>.iso" MD5

например (Рис. 2):

certutil -hashfile "C:\Users\pc\Downloads\solar_safeconnect.iso" MD5

![](Solar_SafeInspect_Инструкция_по_проверке_КС_дистрибутива_media\image002.png)

Рис. 2. Пример ввода команды

- для получения контрольной суммы **SHA 256**:
certutil -hashfile "<путь_к_файлу>\<имя_файла>.iso" SHA256

- например:
certutil -hashfile "C:\Users\pc\Downloads\solar_safeconnect.iso" SHA256

4. Нажать **ENTER**;
5. Дождаться окончания выполнения команды (Рис. 3):
CertUtil: -hashfile — команда успешно выполнена.

![](Solar_SafeInspect_Инструкция_по_проверке_КС_дистрибутива_media\image003.png)

Рис. 3. Сообщение об успешном выполнении команды

6. Найти строку с полученным результатом (Рис. 4):
например:

Хэш MD5 C:\Users\pc\Downloads\SafeConnect.iso:

866d9d67e6d4b959f8275319da4f0574

![](Solar_SafeInspect_Инструкция_по_проверке_КС_дистрибутива_media\image004.png)

Рис. 4. Полученный результат

7. Сверить полученный результат с контрольными суммами дистрибутива, обозначенными в релизнотах (Рис. 5).
![](Solar_SafeInspect_Инструкция_по_проверке_КС_дистрибутива_media\image005.png)

Рис. 5. Сверка полученного результата

В случае, если контрольные суммы файла совпадают с контрольными суммами дистрибутива, обозначенными в релизнотах, можно переходить к процессу инсталляции.

В противном случае, несовпадение означает, что целостность дистрибутива была нарушена, необходимо загрузить файл заново и проверить его контрольные суммы, или обратиться в службу сервиса для получения консультации.

# Проверка контрольных сумм дистрибутива при использовании ОС Linux

## Проверка контрольных сумм дистрибутива с использованием утилит md5sum и sha256sum

Для определения целостности дистрибутива при использовании ОС Linux рекомендуется использовать утилиты **md5sum **и** sha256sum**. Эта утилиты входит в состав разных ОС Linux.

Чтобы определить контрольные суммы дистрибутива необходимо:

1. Загрузить образ диска на **локальное хранилище**;
2. Запустить **эмулятор терминала**;
3. В эмуляторе терминала ввести следующие команды:
- для получения контрольной суммы **MD5**:
md5sum <путь_к_файлу>\<имя_файла>.iso

например (Рис. 12):

md5sum /home/bes/Downloads/SafeConnect_1.12.1.iso

4. ![](Solar_SafeInspect_Инструкция_по_проверке_КС_дистрибутива_media\image012.png)
Рис. 12. Пример ввода команды

- для получения контрольной суммы **SHA256**:
sha256sum <путь_к_файлу>\<имя_файла>.iso

- например (Рис. 13):
sha256sum /home/bes/Downloads/SafeConnect_1.12.1.iso

5. ![](Solar_SafeInspect_Инструкция_по_проверке_КС_дистрибутива_media\image013.png)
Рис. 13. Пример ввода команды

6. Нажать **ENTER**;
7. Дождаться окончания выполнения команды;
8. Найти строку с полученным результатом:
Например (Рис. 14, Рис. 15):

 - 866d9d67e6d4b959f8275319da4f0574/home/bes/Downloads/SafeConnect_1.12.1.iso
 - 9dc6756f63918cf4946b42761ad118c35a808fe6fb6f2f6d0434abc57cd4e203 /home/bes/Downloads/SafeConnect_1.12.1.iso
![](Solar_SafeInspect_Инструкция_по_проверке_КС_дистрибутива_media\image014.png)

Рис. 14. Результат с MD5

![](Solar_SafeInspect_Инструкция_по_проверке_КС_дистрибутива_media\image015.png)

Рис. 15. Результат с SHA256

9. Сверить полученный результат с контрольными суммами дистрибутива, обозначенными в релизнотах (Рис. 16).
В случае, если контрольные суммы файла совпадают с контрольными суммами дистрибутива, обозначенными в релизнотах, можно переходить к процессу инсталляции.

В противном случае, несовпадение означает, что целостность дистрибутива была нарушена, необходимо загрузить файл заново и проверить его контрольные суммы, или обратиться в службу сервиса для получения консультации.
